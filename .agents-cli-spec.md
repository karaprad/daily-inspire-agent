# Agent Spec

## Overview
The **Daily Inspiration Mailer** is an ambient concierge agent that generates and
emails a unique motivational story for Indian kids (ages 10-14) every day. It runs
on a daily schedule with no interactive user, ensuring no story ever repeats by
maintaining a deduplication ledger in Google Cloud Firestore.

## Example Use Cases

### Use Case 1: Daily Scheduled Story
**Input**: Cloud Scheduler triggers at 7:00 AM IST
**Expected Output**: A new, unique motivational story is generated, saved to Firestore,
and emailed to the configured recipient.

### Use Case 2: Manual Trigger
**Input**: User sends "Generate and send today's motivational story"
**Expected Output**: Same as above — a fresh story about an Indian kid doing a good deed.

### Use Case 3: Deduplication
**Input**: Agent is triggered for the 50th time
**Expected Output**: Story #50 is completely different from the previous 49 stories.
The agent checks Firestore for all past titles and morals before generating.

## Tools Required
1. **Firestore** — Read/write story ledger. Uses `google-cloud-firestore` SDK. Auth via service account or ADC.
2. **Gmail SMTP** — Send emails. Uses Python `smtplib`. Auth via Gmail App Password.
3. **Gemini LLM** — Story generation. Uses Google ADK with Vertex AI. Auth via ADC or API key.

## Constraints & Safety Rules
- All content MUST be appropriate for children aged 10-14.
- Stories must be positive, uplifting, and culturally relevant to India.
- No violence, horror, discrimination, or adult themes.
- No PII exposure in story content.
- Prompt injection attempts must be rejected.
- Email credentials must never appear in logs or story content.

## Success Criteria
1. Each story is unique (no duplicate titles or morals across 30+ runs).
2. Email is delivered successfully with correct HTML formatting.
3. Story is saved to Firestore with content hash.
4. Agent completes within 60 seconds per run.
5. Safety rules are enforced (no inappropriate content generated).
